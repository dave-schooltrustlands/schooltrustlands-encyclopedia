# America's School Trust Library

An evidentiary archive of America's school trust lands and funds. Web publication carrying the fifty-state encyclopedia *Schools of the Republic* v1.3 (Sullivan & Bird, 2025), the *Sacred Compact* white paper, the Atlas, the Counting House, and a growing bibliography of foundational works.

A project of the Advocates for School Trust Lands (ASTL), a national 501(c)(3) nonprofit.
